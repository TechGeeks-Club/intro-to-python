2.2. Data Types & Variables#

Primitive Data Types in Python:#

  • Integer (int): Whole numbers, both positive and negative, without decimals.

    • Example: 5, -3, 100

  • Float (float): Numbers with decimal points.

    • Example: 3.14, -2.5, 0.1

  • String (str): A sequence of characters (text), enclosed in either single or double quotes.

    • Example: "Hello", 'Python'

  • Boolean (bool): Represents either True or False.

    • Example: True, False

  • None (NoneType): Represents the absence of a value or a null value.

    • Example: None

How to Define Variables ?#

In Python, defining a variable is simple. You can assign a value to a variable using the = operator. Python automatically infers the type based on the value assigned.

age = 50             # Integer (int)
height = 1.5         # Float (float)
name = "amar"        # String (str)
is_student = True    # Boolean (bool)
favorite_food = None # NoneType (None)

print("Name:", name)
print("Age:", age)
print("Height:", height)
print("Is Student:", is_student)
print("Favorite Food:", favorite_food)  

Name: amar
Age: 50
Height: 1.5
Is Student: True
Favorite Food: None

Tip

you can change the type or the value of the variable any time with any type python     x = 10 # int     x = "hello" #string

Type Conversion (Casting)#

Sometimes, you may want to convert one data type to another. In Python, this can be done using type casting. Here’s how to do it:

#1. convert int to float
x = 3.14
x_int = int(x)  # Converts to integer (3)
print(x_int)

# convert int to string
z = 10
z_str = str(z)  # Converts to string ("10")
print(z_str)

# convert string to boolean
a = "abc"

print(a_bool)

2.3 Operators#

Arithmetic Operators#

Used to perform basic mathematical operations.

  • + (Addition): Adds two numbers.

  • - (Subtraction): Subtracts the second number from the first.

  • * (Multiplication): Multiplies two numbers.

  • / (Division): Divides the first number by the second.

  • // (Floor Division): Divides and returns the largest integer less than or equal to the result.

  • % (Modulus): Returns the remainder of a division.

  • ** (Exponentiation): Raises the first number to the power of the second.

Examples:

a = 10
b = 3
print("Addition:", a + b)         # 13
print("Subtraction:", a - b)      # 7
print("Multiplication:", a * b)   # 30
print("Division:", a / b)         # 3.333...
print("Floor Division:", a // b)  # 3
print("Modulus:", a % b)          # 1
print("Exponentiation:", a ** b)  # 1000

Addition: 13
Subtraction: 7
Multiplication: 30
Division: 3.3333333333333335
Floor Division: 3
Modulus: 1
Exponentiation: 1000

Comparison Operators#

Used to compare two values and return a boolean (True or False).

  • == (Equal): Checks if two values are equal.

  • != (Not Equal): Checks if two values are not equal.

  • >(Greater Than): Checks if the first value is greater than the second.

  • <(Less Than): Checks if the first value is less than the second.

  • >= (Greater Than or Equal To).

  • <= (Less Than or Equal To).

Examples:

x = 5
y = 10
print("x equals y:", x == y)      # False
print("x not equals y:", x != y)  # True
print("x greater than y:", x > y) # False
print("x less than y:", x < y)    # True

x equals y: False
x not equals y: True
x greater than y: False
x less than y: True

Logical Operators#

Used to combine conditional statements.

  • and: Returns True if both conditions are true.

  • or : Returns True if at least one condition is true.

  • not: Reverses the boolean value.

Examples:

a = True
b = False
print("a and b:", a and b) # False
print("a or b:", a or b)   # True
print("not a:", not a)     # False

a and b: False
a or b: True
not a: False

Tip

you can use special assignment operators like += , -=, *=, /=, %=, //=, **=

x = 5
x += 3  # Same as x = x + 3
print("x after += 3:", x) # 8
x *= 2  # Same as x = x * 2
print("x after *= 2:", x) # 16

2.7 Funcions#

A function is a block of code that performs a specific task. Instead of writing the same code multiple times, you can define a function and call it whenever needed. They help to organize and reuse code efficiently.

Syntax :

def function_name(parameters):
    # Code block (body of the function)
    return value  # Optional

Exemples :

# Basic function
def say_hello():
    print("Hello, welcome to the Python workshop!")
    
say_hello() # Output: Hello, welcome to the Python workshop!


# Function with parameters
def hello(name):
    print("Hello,", name)

hello("amar") # Output: Hello, amar


# Function with return value
def add(a, b):
    x = a + b
    return x

result = add(5, 3)
print(result) # Output: 8



# Function with default parameter
def hello_2(name="Guest"):
    print("Hello,", name)

Ex2: Grade Average Calculator#

Objective: Create a grade average calculator for a module. The calculator should ask the user if the module contains TP (practical work). If yes, then the TP and TD (tutorial) are 20% each, and the exam is 60%. Otherwise, the TD is 40% and the exam is 60%. The calculator should then ask the user for their grades in the exam, TD, and TP (if applicable), and calculate the final grade. Note that the grades should be between 0 and 20.

Example: If the module contains TP and the user enters the following grades:

  • Exam: 15

  • TD: 18

  • TP: 17.5

The final grade will be calculated as: Final Grade = (Exam * 0.6) + (TD * 0.2) + (TP * 0.2)
